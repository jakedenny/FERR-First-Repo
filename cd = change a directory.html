cd = change a directory ls = list the files pwd = print working directory (not a password!) mkdir [dirname] = make a directory touch [filename] = create a file rm = remove a file brew install [program] = installs a program (needs brew to be installed) brew doctor brew help git init = Initializes a directory as a git repository git clone [path from github] [dirname] = Clones a git repo git commit -am "A message" = Saves a milestone git push origin master = Saves a cloud milestone git pull origin master = Downloads a cloud milestone The flow will always be: cd to a git repo
